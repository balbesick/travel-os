# travel-os

Приложение, операционная система путешественника.
Одна платформа сопровождает человека от идеи поездки до возвращения домой.

Приложение снижает порог входа в путешествия.

Миссия: Мы делаем международные путешествия понятными для людей, которые никогда раньше не путешествовали самостоятельно.
Мы превращаем сложное путешествие в понятный пошаговый план.

Problem Statement: "Мне для первой поездки в Европу было вообще непонятно, как всё реализовать, но мне повезло найти нужных людей."


MVP 1.0:
* Регистрация и личный кабинет.
* Создание путешествия.
* Пошаговый Roadmap подготовки.
* Чек-лист документов.
* Планировщик маршрута.
* Календарь ключевых событий (покупка билетов, подача на визу и т.д.).
* Планировщик бюджета.
* Уведомления.
